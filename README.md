Профессиональный рандомизатор текста и генератор объявлений от Айрата Халитова (десктоп версия)


Создать Makefile (в первый раз):

```
qmake
```

Скомпилировать проект:

```
make
```

Запустить программу:

```
./ad-generator-desktop &
```

Или использовать скрипт:

```
./run.sh
```
